# EduSpectro4X
ファームウェア・電子回路
[📖 総合ガイド（Webサイト）はこちら](https://eduspectro.github.io/)


*   **概要**: 「RP2350とC12880MAを組み合わせた分光器の心臓部です」
*   **特徴**: PIOによる高速制御、Catmull-Rom補間など（公式サイトの内容を要約）。
*   **必要な環境**:
    *   Arduino IDEのバージョン
    *   導入が必要なボードマネージャー（Raspberry Pi Pico/RP2350）
    *   依存ライブラリ（あれば）
*   **ハードウェアについて**:
    *   「KiCadの使い方については `docs/KiCad_Guide.pdf` を参照してください」という一言。
*   **使い方**: ファームウェアの書き込み方法。
